# الخلفية النظرية — تنفيذ الاستراتيجية وإدارة مؤشرات الأداء

<div dir="rtl">

## كيف ترتبط هذه الموضوعات بمنصة رافد KPI

```
موضوع نظري                         ←    كيف يتجلى في رافد KPI
──────────────────────────────────────────────────────────────────────
تنفيذ الاستراتيجية (01)            ←    التسلسل الهرمي لأنواع الكيانات (ركائز → أهداف → مؤشرات)
تصميم مؤشرات الأداء (02)           ←    حقول الكيان: الاتجاه، الدورية، الوحدة، الصيغة، الهدف
الأطر (03)                         ←    أنواع لوحات المتابعة: التنفيذي، الأداء، الحوكمة
الحوكمة (04)                       ←    سير العمل: مسودة → إرسال → اعتماد → قفل
الصحة ونظام RAG (05)               ←    مقاييس درجات الصحة، الشارات اللونية، بطاقات إحصاءات النظرة العامة
```

## ترتيب القراءة الموصى به

1. ابدأ بـ **01 — أسس تنفيذ الاستراتيجية** لفهم الصورة الكبيرة.
2. اقرأ **02 — تصميم مؤشرات الأداء** لفهم كيفية هيكلة مؤشرات الأداء الفردية.
3. راجع **03 — الأطر** لرؤية كيفية تعامل مؤسسات مختلفة مع إدارة الأداء.
4. ادرس **04 — الحوكمة** لفهم سبب وجود سير عمل الاعتماد.
5. اختتم بـ **05 — الصحة ونظام RAG** لفهم كيفية تصاعد مؤشرات الأداء الفردية إلى درجات مؤسسية.
