## ملخص البدء السريع

1. **سجّل دخولك** على `/<locale>/auth/login` ببيانات اعتماد مؤسستك.
2. صفحة الهبوط هي **النظرة العامة** — لوحة ملخص الصحة المؤسسية المخصصة لك.
3. تنقّل باستخدام **الشريط الجانبي الأيسر**: النظرة العامة ← الكيانات (مؤشرات الأداء) ← الاعتمادات ← لوحات المتابعة.
4. يُحدَّد مستوى وصولك بناءً على **دورك** (مسؤول، أو تنفيذي، أو مدير).
5. تمر بيانات مؤشرات الأداء بدورة عمل **مسودة ← إرسال ← اعتماد** قبل أن تُحتسب رسمياً.

## مفاهيم النظام بنظرة سريعة

```
المؤسسة (Organization)
 └── أنواع الكيانات  (مثل: الاستراتيجية، الأهداف، مؤشرات الأداء)
      └── الكيانات  (السجلات الفردية)
           └── قيم الكيانات  (القياسات الدورية)
                └── دورة الاعتماد  (مسودة ← مُرسَل ← معتمد/مقفل)
```

- **الكيانات** هي لبنات البناء القابلة للتهيئة — تحدد كل مؤسسة أنواعها الخاصة (مثل: الركيزة الاستراتيجية، المبادرة، مؤشر الأداء).
- **قيم الكيانات** هي نقاط البيانات الفعلية (القياسات الرقمية) التي يُدخلها المستخدمون على الكيانات بمرور الوقت.
- **الاعتمادات** تتحكم في القيم التي تصبح رسمية — استناداً إلى مستوى الاعتماد المُهيَّأ على مستوى المؤسسة.

## اللغات المدعومة

يدعم النظام **اللغة العربية** (من اليمين إلى اليسار) و**اللغة الإنجليزية** (من اليسار إلى اليمين). غيّر اللغة عبر بادئة اللغة في الرابط (`/ar/...` للعربية، `/en/...` للإنجليزية) أو من خلال زر تبديل اللغة في واجهة المستخدم.
