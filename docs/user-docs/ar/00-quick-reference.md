# الدليل السريع

<div dir="rtl">

مرجع سريع لأهم المهام في نظام  المؤشرات KPI.

---

## تسجيل الدخول

```
/ar/auth/login
```

1. أدخل بريدك الإلكتروني
2. أدخل كلمة المرور
3. انقر **تسجيل الدخول**

---

## إدخال قيمة مؤشر أداء

### المسار السريع:

1. انتقل إلى **الكيانات** ← [نوع الكيان]
2. انقر على الكيان المطلوب
3. انتقل إلى تبويب **القيم**
4. انقر **إضافة قيمة جديدة**
5. أدخل القيمة والفترة
6. انقر **حفظ كمسودة**
7. عند الجاهزية، انقر **إرسال للاعتماد**

---

## الاعتماد على قيمة مؤشر أداء

### للمعتمدين:

1. انتقل إلى **الاعتمادات** في الشريط الجانبي
2. اختر القيمة المراد اعتمادها
3. راجع القيمة والملاحظات
4. انقر **اعتماد** أو **رفض**

---

## عرض لوحات المتابعة

### المسارات المختصرة:

| اللوحة | الرابط |
|--------|--------|
| النظرة العامة | `/ar/overview` |
| التنفيذية | `/ar/dashboards/executive` |
| PMO | `/ar/dashboards/pmo` |
| المدير | `/ar/dashboards/manager` |
| أداء KPI | `/ar/dashboards/kpi-performance` |

---

## إنشاء مستخدم جديد (للمسؤول)

```
/ar/users → جديد
```

**الحقول المطلوبة:**
- الاسم
- البريد الإلكتروني
- الدور (EXECUTIVE, MANAGER, ADMIN)
- الإدارة

---

## تكليف مستخدم بكيان (للمسؤول)

```
/ar/responsibilities → حسب الكيان → تكليف
```

1. اختر الكيان
2. انقر **تكليف**
3. اختر المستخدم (المستخدمين)
4. انقر **تأكيد**

---

## تصدير تقرير

### CSV/Excel:

1. انتقل إلى **التقارير**
2. اختر نوع التقرير
3. طبق الفلاتر المطلوبة
4. انقر **تصدير** ← اختر التنسيق

---

## تغيير اللغة

### يدوياً:

غيّر `ar` إلى `en` في الرابط:

```
/ar/overview    → الواجهة العربية
/en/overview    → الواجهة الإنجليزية
```

---

## الاختصارات المفيدة

| الاختصار | الوظيفة |
|----------|---------|
| `/` | التركيز على البحث |
| `?` | فتح المساعدة |
| `Esc` | إغلاق النافذة المنبثقة |
| `Ctrl+Enter` | حفظ النموذج |
| `Ctrl+Shift+R` | إعادة تحميل مع مسح الذاكرة المؤقتة |

---

## حلول سريعة للمشاكل

| المشكلة | الحل السريع |
|---------|-------------|
| الصفحة لا تُحمّل | F5 لإعادة التحميل |
| بيانات قديمة | Ctrl+Shift+R |
| خطأ في الصيغة | اختبر الصيغة بـ "تشغيل الاختبار" |
| لا يمكن الإرسال للاعتماد | تأكد من ملء جميع الحقول المطلوبة |
